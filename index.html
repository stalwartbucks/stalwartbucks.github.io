Stalwartbucks [SBX]

Stalwartbucks is a new cryptocurrency brought to you by Joe Weisenthal. Stalwartbucks is a clone of Dogecoin, which in turn is based on Litecoin. Guan Yang did the search-and-replacing.

Get it

  • Mac client (on Mountain Lion and Mavericks, right-click on the app and click Open)
  • Source code
  • We are working on a Windows binary (does anyone still use Windows?) (has anyone built it successfully?)
  • Command line stalwartbucksd for Windows (all the same features as graphical wallet, with more typing)
  • Linux users: build from source, nerds!

Services

  • Blockchain
  • Pool (alpha)
  • Guide to mining with the Pool

Details

Proof of work: scrypt

To make it easier for everyone to become a billionaire, each raw coin is designated as a Billion Stalwartbucks [B SBX].

Block reward starts at 1,000 trillion SBX (1,000,000 raw coins) for the first 10,000 blocks and falls gradually to 1 trillion SBX (1,000 raw coins).

Premined by the creators, for ~1,500 blocks.
